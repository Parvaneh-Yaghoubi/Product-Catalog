# Product Catalog

A Multi-Page application that allows users to explore products.

---

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Parvaneh-Yaghoubi/Product-Catalog.git
cd Product-Catalog
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start the development server

```bash
npm run dev
```

### 4️⃣ Open in browser

Visit:

```
http://localhost:3000
```

or the port shown in your terminal.
